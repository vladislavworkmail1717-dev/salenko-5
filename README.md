# Salenko-5 Website

Шаблон сайту, який був створений як домашнє завдання.

## Опис проєкту

Сайт демонструє базову структуру з використанням HTML та CSS.  
Він містить навігацію між сторінками, зображення та стилізацію елементів.

## Структура проєкту

- `index.html` — головна сторінка сайту
- `style.css` — стилі сайту
- `img` — папка із зображеннями
- `pages` — додаткові сторінки сайту
